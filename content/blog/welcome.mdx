# 欢迎来到我的新博客 🎉

经过几周的开发和设计，我的全新博客平台终于上线了！这次重构采用了最新的技术栈，带来了全新的体验。

## ✨ 新特性

### 1. 现代化的技术栈

- **Next.js 14**: 使用最新的 App Router，提供更好的性能和开发体验
- **React 18**: 利用最新的 React 特性，包括 Server Components
- **TypeScript**: 类型安全，更好的开发体验
- **Tailwind CSS**: 现代化的样式方案，快速开发精美UI

### 2. 优秀的用户体验

- 🎨 **精美的设计**: 参考大厂设计规范，打造专业级UI
- 🌓 **深色模式**: 支持自动切换，保护你的眼睛
- 📱 **响应式设计**: 完美适配所有设备
- ⚡ **极致性能**: 静态生成，秒开体验

### 3. 强大的工具集

除了博客功能，还集成了多个实用工具：

- 📊 成交量分析器
- 🧭 AI 命理决策系统
- 🎨 JSON 格式化工具
- 🔍 正则表达式测试器

## 📝 关于博客内容

这个博客将分享：

1. **技术文章**: 前端、后端、工程化等技术实践
2. **工具开发**: 分享各种实用工具的开发过程
3. **量化分析**: 股票、数据分析相关内容
4. **AI 应用**: 人工智能相关的应用和实践
